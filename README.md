# Tower Defense

## Unity Version: 6000.0.31f1

### Controls  
- **Movement**  
  - `WASD` → Move the player on the ground plane.  
- **Combat**  
  - Player automatically attacks the closest enemy within range.  
- **Waves**  
  - `Space` → Call the next wave early.  
  - **Next Wave** button in the UI can also trigger it.  
- **Pause**  
  - `Escape` → Toggle pause/unpause.  
- **Debug Shortcuts**  
  - `F1` → Show waypoints and gizmos.  
  - `F2` → Destroy all active enemies.  
  - `F3` → Show spawn points.  
  - `Z` → Increase TimeScale (2x).  
  - `-` (Minus) → Decrease TimeScale (0.5x).  
  - `0` (Alpha0) → Reset TimeScale (1x).
 
### Limitations
- All art, audio, and visual assets are **placeholder** for prototyping purposes.
